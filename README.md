# PiBlox
A program for running roblox studio with wine and box86.
# Install
Run this command:

```cd $HOME && wget -qO- https://raw.githubusercontent.com/MakePie/PiBlox/main/install | bash```

# Install Studio
Run this:

```bash $HOME/.piblox/InstallStudio```
# Run Studio
Run this:

```bash $HOME/.piblox/RunStudio```
